Work Projects
Autodesk Forge Viewer

This project is an adaptation of Autodesk Forges' CAD viewer. By default, the Forge viewer handles BIM and DGN files, and has a wide array of customisable features, including extensions. I customised and maintain this viewer at work. :)

Additionally, the code for this is private, but it is similar in appearance to Autodesks' online viewer, which can be seen here. A (free) account on Autodesk is required to view this!

Potree Point Cloud Viewer

This project is also an adapation, this time of an open-source WebGL based point cloud renderer known as Potree. Similarly, the potree viewer has a wide range of customisation available and I maintain this at work. :)

Again, this code is private, but examples of how this looks in browser can be found on Potree's github repository.
